# School Management App - Complete Color Theme Guide

Based on your landing page design, here's a comprehensive color theme for your school management application.

## üé® Core Color Palette

### Primary Colors
- **Primary Blue**: `#003DFF` - Main brand color for buttons, links, active states
- **Primary Dark**: `#0029CC` - Hover states, pressed buttons
- **Primary Darker**: `#001A99` - Active/pressed states

### Accent Colors
- **Accent Teal**: `#00D4AA` - Success states, positive indicators, highlights
- **Accent Cyan**: `#00B8E6` - Info states, secondary highlights

### Background Colors
- **Main Background**: `#FFFFFF` - Primary content area
- **Secondary Background**: `#F8F9FA` - Cards, panels, alternate sections
- **Tertiary Background**: `#F0F2F5` - Subtle dividers, hover states

### Text Colors
- **Primary Text**: `#0A2540` - Main headings, important text
- **Secondary Text**: `#6B7280` - Body text, descriptions
- **Tertiary Text**: `#9CA3AF` - Placeholders, disabled text
- **White Text**: `#FFFFFF` - Text on dark backgrounds

### Status Colors
- **Success**: `#00D4AA` (Accent Teal) - Completed tasks, positive actions
- **Error**: `#EF4444` - Errors, warnings, delete actions
- **Warning**: `#F59E0B` - Warnings, pending states
- **Info**: `#00B8E6` (Accent Cyan) - Information, notifications

## üí° Design Principles

1. **Consistency**: Use the same color palette throughout the app
2. **Contrast**: Ensure WCAG AA compliance (4.5:1 for text)
3. **Hierarchy**: Use color to establish visual hierarchy
4. **Accessibility**: Provide sufficient contrast for all users
5. **Brand Identity**: Maintain connection with landing page colors

---

## üé® Color Usage Guidelines

### Do's ‚úÖ
- Use primary blue (#003DFF) for main actions and active states
- Use accent teal (#00D4AA) for success and positive feedback
- Use dark blue (#0A2540) for sidebar and important sections
- Use white/light backgrounds for content areas
- Use gray scale for secondary information

### Don'ts ‚ùå
- Don't use too many colors at once
- Don't use colors that clash with the primary palette
- Don't use low contrast combinations
- Don't use pure black (#000000) - use dark blue instead
- Don't use bright colors for large backgrounds

---

## üì± Responsive Considerations

- Mobile: Lighter backgrounds, larger touch targets
- Tablet: Maintain desktop color scheme
- Desktop: Full color palette with hover states
